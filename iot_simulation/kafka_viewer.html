📋 Commandes pour récupérer les messages

Ouvrez un terminal (cmd) et exécutez ces commandes :


Pollution:
docker exec -it kafka kafka-console-consumer --topic smartcity-pollution --bootstrap-server localhost:9092 --from-beginning --max-messages 5

Trafic:
docker exec -it kafka kafka-console-consumer --topic smartcity-traffic --bootstrap-server localhost:9092 --from-beginning --max-messages 5

Éclairage:
docker exec -it kafka kafka-console-consumer --topic smartcity-lighting --bootstrap-server localhost:9092 --from-beginning --max-messages 5

Déchets:
docker exec -it kafka kafka-console-consumer --topic smartcity-waste --bootstrap-server localhost:9092 --from-beginning --max-messages 5

💡 Astuce: Pour voir les messages en temps réel (flux continu), supprimez --from-beginning --max-messages 5 et appuyez sur Ctrl+C pour arrêter.

🎯 Pour la soutenance: Affichez ce dashboard dans le navigateur ET les commandes ci-dessus dans un terminal.
